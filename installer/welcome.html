Welcome to Mirror Mirror

This installer will:

  • Install MirrorMirror.vst3 and MirrorMirror.component to your Audio Plug-Ins folder
  • Set up the local AI server that powers the plugin
  • Configure the server to start automatically at login

Note: After installation, the server will download its Python dependencies in the background (~500 MB). This takes a few minutes on first launch. Progress is logged to:
~/Library/Application Support/MirrorMirror/install.log

Once the server is ready, open your DAW and scan for new plugins — Mirror Mirror will appear.
